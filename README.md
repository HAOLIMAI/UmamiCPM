# UmamiCPM
This project implements a stacking ensemble framework for predicting umami peptides. The model is trained on the Pep318 dataset and validated on an independent test set. Three meta‑models — Logistic Regression, Random Forest, and GBDT — are tuned via GridSearchCV with 5‑fold cross‑validation.

Before running the script, please install the required Python packages (Python 3.7+ recommended) by executing:
pip install pandas scikit-learn openpyxl
